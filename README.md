## About
This repository presents a test stand, JMH benchmarks, profiling scripts, and experimental artifacts for comparing system time determinization methods in Java unit testing.

The project is completed during the preparation of Salavat I. Yabbarov's work at SPbPU Institute of Computer Science and Cybersecurity (SPbPU ICSC).
